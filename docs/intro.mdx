# 文档首页

这里是一个以技术内容为主的个人站点，方向聚焦在：

- 后端工程
- 云原生 / Kubernetes
- JVM 与 Java 生态
- Linux 运维与排障
- LLM 应用落地

## 这个站点会写什么

### 博客区

博客区更偏日常技术输出，适合放：

- 排障记录
- 架构理解
- 性能调优经验
- 技术方案对比
- 项目复盘
- 对某个热点技术的工程化理解

### 文档区

文档区更偏体系化整理，适合放：

- 专题索引
- 知识地图
- 操作手册
- 最佳实践清单
- 长期维护的技术笔记

## 写作风格

我更希望这里的内容是“正常工程师会收藏和转发的技术博客”，所以会尽量坚持几件事：

- 少空话，多写清楚为什么
- 少堆概念，多给判断标准
- 少复制官方文档，多补真实经验
- 少只讲 happy path，多写排障思路

## 从哪里开始看

如果你是第一次来，可以先：

- 先看《[Kubernetes 专题总导航](/docs/kubernetes-index)》
- 去博客区看最新文章
- 从 Kubernetes 和云原生相关文章开始
- 再按自己关心的方向看 JVM、Linux 和 LLM 主题
