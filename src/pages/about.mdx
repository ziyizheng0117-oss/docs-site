# About

这是一个偏实战的技术博客，主要写：

- 后端开发
- 云原生、Kubernetes
- JVM 与 Java 生态
- Linux 运维、排障
- LLM 应用落地

我希望这里的内容尽量像一个正常工程师会长期维护的技术博客，而不是只做信息搬运。

所以会更关注这些东西：

- 工程实践里真正会遇到的问题
- 架构设计背后的取舍
- 性能优化和稳定性治理
- 线上故障排查的思路
- 新技术落地时的收益、成本和坑

如果你喜欢“少一点空话，多一点经验；少一点概念堆砌，多一点判断标准”的技术内容，这里应该会比较对路。

你可以从这些方向开始看：

- 博客：看最新文章和排障记录
- 文档：看专题整理和知识索引
