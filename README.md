# 小屈同学的技术博客

一个基于 Docusaurus 搭建并部署到 GitHub Pages 的个人技术博客。

## 博客定位

这个博客主要写：

- 后端开发
- 云原生 / Kubernetes
- JVM 与 Java 生态
- Linux 运维与排障
- LLM 应用落地

内容会更偏：

- 工程实践
- 架构设计
- 性能调优
- 稳定性治理
- 问题排查与复盘

目标不是做一份“官方文档搬运站”，而是尽量把真实项目里会遇到的问题、判断和经验写清楚。

## 本地开发

```bash
npm install
npm start
```

## 构建

```bash
npm run build
```

## 内容位置

- `docs/`：专题文档、索引、结构化知识
- `blog/`：博客文章、实践记录、排障复盘
- `src/pages/about.mdx`：站点 About 页面

## 部署

本项目已配置为 GitHub Pages 静态发布。

## 仓库建议描述

专注后端、云原生、JVM、Linux 与 LLM 的技术博客，记录工程实践、架构设计、性能优化和问题排查。
